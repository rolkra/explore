# Plot a text

Plots a text (base plot) and let you choose text-size and color

## Usage

``` r
plot_text(text = "hello world", size = 1.2, color = "black", ggplot = FALSE)
```

## Arguments

- text:

  Text as string

- size:

  Text-size

- color:

  Text-color

- ggplot:

  return a ggplot-object? (or base plot)

## Value

Plot

## Examples

``` r
plot_text("hello", size = 2, color = "red")
```
